# GOAL — hexa-matter 의 한 문장

```
/goal hexa-matter 가 — 중국 정제 독점(~90%)에 묶인 희토류 영구자석을 n=6
격자수에 끼워맞춘 종이 대체재나 시뮬레이션 예측을 측정으로 둔갑시킨 주장으로
"해결됐다" 하지 않고 — REE-free 자석 후보(hxm-mag-*)를 정량 falsifier 와 함께
DESIGN → SIM-DFT/NNP → SYNTH-ROUTE 로 정직하게 심화시켜, 오직 외부 wet-lab
증거로만 EXTERNAL-VERIFIED 에 이르는 반증가능한 탈희토류(de-rare-earth)
경로로 만든다.
```

> **한 문장 (canonical)**: hexa-matter 가, 희토류 의존 영구자석을 격자
> 편의수에 끼워맞춘 종이 대체재나 예측을 측정으로 둔갑시킨 주장으로 "해결"
> 선언하지 않고, REE-free 자석 후보(`hxm-mag-*`)를 정량 falsifier 와 함께
> `DESIGN → SIM → SYNTH-ROUTE` 로 정직하게 심화시켜 **오직 외부 wet-lab
> 증거로만** `EXTERNAL-VERIFIED` 에 이르는 *반증가능한* 탈희토류 경로로 만든다.

## 무엇이 아닌가 (NOT)

- n=6 격자·편의수에 물성을 끼워맞춘 후보 아님 (`@F f1` lattice-fit-on-external-entities)
- DFT/NNP/ML 예측을 측정값으로 둔갑시킨 주장 아님 (`@F f2` predictions-promoted-to-measurements)
- 소프트웨어가 후보를 스스로 `EXTERNAL-VERIFIED` 로 승급 아님 (`@F f5` category-c-self-promotion)
- UNPROVEN/UNVERIFIED 마커를 지운 "깨끗해 보이는" 후보 아님 (`@F f4` unproven-marker-stripped)
- 격자 적합도로 프로젝트 천장을 정하는 것 아님 (real-limits-first, `@D g1`)

## 무엇인가 (IS)

- 정량 falsifier 를 달고 — 실험으로 깨질 수 있는 — 후보 (`@D g9` novel-ledger-discipline)
- `DESIGN → SIM-DFT → SIM-NNP → SYNTH-ROUTE` 단계마다 거리가 측정되는 후보 (NOVEL.md §2 pipeline)
- 격자(n=6)는 도구일 뿐, 천장은 실제 자기물리 (Stoner-Wohlfarth · Brown 부등식 · (BH)max ≈ 64 MGOe) (`@D g2`)
- 외부 wet-lab 증거로만 최종 등급에 이르는 후보 (Category (c) out-of-repo)

## 현재 정직한 위치 (g3 — over-claim 금지)

**목표 미도달.** 경로는 열렸고 거리는 측정됐으나 실증 탈희토류 자석은 미발현 — 그게 정직한 현재.

- `hxm-mag-*` roster: 11 후보 — **7 SIM-DFT** (MP cross-ref) + **1 SIM-NNP** (boride: CHGNet relax of C16 ordered approximant, ubu-1 GPU) + 3 DESIGN (ferrhd · mnbi · znfe)

- 6-트랙 중 재활용(TRACK 1)만 상용 근접, 나머지(ferrite-hd · Mn-Al-C · boride · low-Dy/Tb · AI-found)는 실험실/예측 단계

## 템플릿 — 다른 후보군 재사용

`<domain>+ALTERNATIVE` 계열 도메인은 아래 골격을 치환해 자체 GOAL 한 문장을 둔다:

```
/goal <project> 의 <candidate-prefix>-* 후보가 — <convenience-frame> 에
끼워맞춘 숫자나 시뮬레이션 예측을 측정값으로 둔갑시킨 종이 후보가 아니라 —
정량 falsifier 를 단 채 DESIGN → <sim-stage> → SYNTH-ROUTE 를 정직하게
통과해, 오직 외부 <external-evidence> 증거로만 <terminal-grade> 에
도달하는 반증가능(falsifiable) <domain> 후보로 심화하게 만든다.
```

치환 키:

| 키 | 의미 | 본 프로젝트 값 |
|---|---|---|
| `<project>` | repo / 프로젝트 | hexa-matter |
| `<candidate-prefix>` | 후보 ID 접두 | hxm-mag |
| `<convenience-frame>` | 끼워맞추면 안 되는 편의 틀 | n=6 격자 |
| `<sim-stage>` | 시뮬레이션 단계 | SIM-DFT / SIM-NNP |
| `<external-evidence>` | 최종 등급에 필요한 외부 증거 | wet-lab |
| `<terminal-grade>` | 외부 증거로만 도달하는 등급 | EXTERNAL-VERIFIED |
| `<domain>` | 후보군 도메인 | 자석 (magnet) |
